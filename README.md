# 🐍 Python Learning Journey

This repository documents my journey of learning Python. I'm following [CodeWithHarry's Complete Python Course](https://www.youtube.com/watch?v=UrsmFxEIp5k), and as I learn new concepts, I add my notes and practice programs here.

The goal is simple — not just watching the videos, but actually writing code to solidify the concepts, and tracking the whole journey along the way. This might also help someone else who's learning Python.

## 📂 Repo Structure

Each chapter has its own folder containing:
- `notes.md` → a short summary of the concept in my own words
- `practice.py` → solutions to that chapter's practice questions

```

├── README.md
```
